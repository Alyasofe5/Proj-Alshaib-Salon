Typography

Motion & Animations

The platform utilizes performance-friendly CSS animations that trigger on first paint to ensure above-the-fold content appears elegantly without layout shifts.

Keyframe Name Description Timing & Easing Usage
heroFadeUp Fades in and shifts Y from 18px to 0. 0.45s ease (staggered) Headers, Paragraphs, CTAs on page load
heroSlideRight Fades in and shifts X from RTL start. 0.4s ease Hero badges, side elements
glassCardIn Scales from 0.97 to 1, fades in. 0.4s ease (staggered) Dashboard statistic cards
Transition Universal hover state smoothing all 0.3s Buttons, Cards, Links
